**MapasApp** es una aplicación Angular 14 que sirve como demostración educativa de las capacidades de **Mapbox GL** (v2.9.2) para crear mapas interactivos. La aplicación presenta cuatro módulos principales: (1) **FullScreen** - visualización básica de un mapa a pantalla completa centrado en Gijón, España; (2) **ZoomRange** - controles interactivos de zoom con slider y botones, además de visualización en tiempo real de las coordenadas del centro del mapa; (3) **Marcadores** - un sistema completo de gestión de marcadores donde los usuarios pueden crear, arrastrar y eliminar marcadores de colores en el mapa, con persistencia de datos en LocalStorage para que los marcadores se mantengan entre sesiones; y (4) **Propiedades** - una página que muestra un listado de propiedades inmobiliarias (casas y departamentos en diferentes países) cada una con un mini-mapa embebido mostrando su ubicación. La aplicación está estructurada con componentes reutilizables, utiliza Bootstrap para el diseño responsivo, y demuestra conceptos clave como binding de datos entre UI y estado del mapa, manejo de eventos de Mapbox, y persistencia de datos en el navegador.
